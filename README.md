# Wireless Energy Transfer and Obstacle-Based Steering Lock System

## Overview
Wireless energy transfer–based vehicle safety system using *ESP32* and *ultrasonic sensors* to detect obstacles and automatically activate a *steering lock. Implemented in **Embedded C* with *IoT-based real-time monitoring*.
